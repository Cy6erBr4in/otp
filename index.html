    The branch in its current shape cannot be included in OTP because it includes copyrighted code with an unknown license. It is also overkill to emulate mmap() (and it could cause other problems in the future); the easiest and most straightforward solution is to implement mseg_create()/mseg_destroy() (in erl_mseg.c) using VirtualAlloc()/VirtualFree().
